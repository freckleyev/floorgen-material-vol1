General Information

The Material Pack is provided as a .ZIP file and can be easily installed into the add-on’s Material Library. Please refer to the manual page for the setup process. Once installed, all materials are automatically organized within the library.

An Asset Library version of this Material Pack will be released in a future update.

The number of materials included in this pack will continue to grow with upcoming updates.

Full list of materials in Material Pack Vol. 1

Below is the list of all the materials with rendered previews. Every item has a description including recommended size of the board, type of the material, texture count for MultiTexture and texture dimensions (dimensions per each board for MultiTexture material and dimensions of the seamless texture for Seamless material).

Each texture used in MultiTexture is optimized to be approximately 1 MB or less.
